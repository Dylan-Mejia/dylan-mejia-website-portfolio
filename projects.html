Web Database With Game Users

This project invloves a website coded with PHP and connects with MySQL to a database, allowing administrators to sign up with a username, email, and password. Input vailidation is put in place to ensure the correct information is inputted. Once they are registered they have the option to view the database records as well as adding new records, searching, and removing records. The records are themed as a video game profile database, with player IDs and other information like usernames, emails, hours played, XP Levels and the year joined. This website demonstrates how an adminstrator account would be created to manage data in this database.

Password Security

These are 3 Python scripts that showcase the use of password security. One of them is a password generator that generates unique and strong passwords to users. The second one is a password manager that lets the user create an account by inputting a username and a password. The script can store the password as a hash and the user also has the option to test their username and password login. The thrid script is a password manger that takes user input and gives the strength of the password on a scale of 1 to 5. 1 being very weak and 5 being very strong. The checker looks for all types of character usage like numbers, special chars, and even whitespace to determine the score.
